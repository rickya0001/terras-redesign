# Terra's Menu — Lightweight HTML

A phone-first, low-bandwidth menu page for Terra's Urban Mexican Kitchen.

## Files

- `index.html` — menu page with minimal inline CSS, no JavaScript, no external fonts, and two small header images from the original site.
- `data/menu.json` — structured scraped menu data.

## Size target

`index.html` is currently about **10.6 KB**, under the 14 KB design goal. Main menu choices are horizontal JavaScript tabs; menu subsections use a small previous/next carousel.

## Preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8080
```
